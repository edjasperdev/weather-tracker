# Weather App
## Get a Five-Day Forcast of your Favorite Cities
======
### By Edward Jasper

* ES6
* SASS
* HTML Webpack Plugin
* Extract Text Plugin
* Tests with Mocha and Chai
* OpenWeatherMap API


### To Run: 
##### npm install
##### npm run start
